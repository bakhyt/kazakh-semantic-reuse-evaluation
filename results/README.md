# Results

This folder contains result tables reported in the CLEF 2026 paper.

## Files

- `pairwise_results.csv`  
  Results for pairwise PAN-KK semantic reuse classification.

- `news_results.csv`  
  Results for external Kazakh news-domain snippet-pair evaluation.

- `long_document_results.csv`  
  Results for long-document semantic reuse detection.

- `retrieval_ablation_results.csv`  
  Results for candidate retrieval ablation experiments.

## Metrics

The reported metrics include:

- accuracy
- precision
- recall
- F1
- runtime where applicable
