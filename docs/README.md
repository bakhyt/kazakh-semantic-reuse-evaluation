# Documentation

This folder contains supporting documentation for the CLEF 2026 camera-ready repository.

## Files

- `camera_ready_notes.md`  
  Notes about repository updates for the camera-ready version.

- `data_availability.md`  
  Explains dataset availability and redistribution limitations.

- `generative_ai_use.md`  
  Declaration of generative AI use.

- `minilm_news_zero_result.md`  
  Explanation of the MiniLM zero precision, recall, and F1 result in the news-domain evaluation.

- `reproducibility_checklist.md`  
  Checklist of repository materials and remaining reproducibility notes.

- `reviewer_response_notes.md`  
  Summary of how the repository addresses reviewer concerns.
