# Sample Input Formats

This folder contains small artificial CSV examples showing the expected input format for the public evaluation scripts.

The examples are synthetic and are not taken from PAN-KK, Nur.kz, Tengrinews.kz, or any copyrighted dataset.

## Files

- `pairwise_sample.csv`  
  Example format for pairwise PAN-KK-style evaluation.

- `news_sample.csv`  
  Example format for external news-domain evaluation.

- `long_document_sample.csv`  
  Example format for long-document evaluation.

These files are provided only to demonstrate column names and input structure.
