## Data Availability

The full datasets are not redistributed in this repository.

This repository provides:

- code templates
- configuration files
- documentation
- result tables
- synthetic sample input formats

PAN-KK was introduced in the authors' previous work. The full PAN-KK dataset is not currently redistributed here.

The external Kazakh news benchmark is based on articles from Nur.kz and Tengrinews.kz. Full news articles are not redistributed because copyright remains with the original publishers.

Synthetic sample CSV files are provided only to demonstrate expected input column names and file structure.

## Reproducibility Note

The scripts in this repository use placeholder paths such as `path/to/...`. Users should replace these with local paths to their own datasets and model checkpoints.
