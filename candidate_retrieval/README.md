# Candidate Retrieval

This folder contains public template scripts for candidate retrieval in the long-document semantic reuse detection pipeline.

## Files

- `tfidf_retrieval.py`  
  TF-IDF cosine similarity candidate retrieval.

- `simhash_retrieval.py`  
  SimHash candidate retrieval with adaptive Hamming-distance thresholds.

- `candidate_retrieval_description.md`  
  Documentation of the candidate retrieval stage.

## Notes

The public scripts use placeholder inputs and do not include private datasets, local paths, or full text outputs.
